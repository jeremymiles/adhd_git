## 2024-05-14 - Use accessible color palettes for data visualizations
**Learning:** Hardcoded basic colors (like "red", "green", "yellow", "chocolate4") in data visualizations like ggplot2 can be extremely difficult to distinguish for users with color vision deficiencies, reducing accessibility.
**Action:** Always replace basic hardcoded colors in plots with accessible, colorblind-friendly palettes such as Okabe-Ito (e.g., `#E69F00`, `#56B4E9`, `#009E73`, `#F0E442`, `#0072B2`, `#D55E00`, `#CC79A7`).
## 2024-05-15 - Add dynamic alt text to ggplot2 in Quarto documents
**Learning:** Screen readers cannot interpret images generated by `ggplot2` in Quarto/RMarkdown documents unless alternative text is explicitly provided. When generating plots inside a loop, static alt text fails to describe the specific contents of each plot.
**Action:** Always include `labs(alt = "...")` in `ggplot2` calls, and when generating plots iteratively within a loop, dynamically construct the alt text (e.g., using `paste()`) to accurately reflect the data being visualized in that specific iteration.
