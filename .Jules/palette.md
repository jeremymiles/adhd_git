## 2024-05-24 - Improve Data Visualization Accessibility with Okabe-Ito Palette
**Learning:** Hardcoding basic colors like "red", "green", and "yellow" for data visualizations makes charts inaccessible for users with color vision deficiencies. It is critical to use colorblind-friendly palettes to ensure readability for all users.
**Action:** When creating or maintaining plots, always default to accessible palettes such as Okabe-Ito (`#E69F00`, `#56B4E9`, `#009E73`, `#F0E442`, `#0072B2`, `#D55E00`, `#CC79A7`) rather than basic named colors.
